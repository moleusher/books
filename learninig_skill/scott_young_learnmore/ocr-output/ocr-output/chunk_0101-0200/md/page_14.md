## Practice, Practice, Practice!

Like all the methods in this book, these techniques require practice to use effectively. If you aren’t used to using them it can take 2-3 times as long to apply them. Furthermore, when you are beginning with these techniques, I suggest making use of a pen and paper. I don’t write out metaphors and diagrams if I can imagine them clearly, but if you aren’t used to these techniques, paper is the best way to practice.
