# Challenges With the Link Method

Although the link method can be more time consuming than other techniques, it is reliable and your speed will improve with practice. There are a few other challenges that can arise with using the link technique you will need to be careful of when creating a list:

## Repetition of Symbols

Let's say you want to remember the following formula sequence:

 $  \mathbf{r}, =, \sigma, (, \mathbf{Z}, \mathbf{x}, *, \mathbf{Z}, \mathbf{y}, ), \text{Divide}, (, \mathbf{n}, \text{Minus}, 1, )  $
