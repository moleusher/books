The right-hand side vector f is given by ([18], (4.14))

 $$ \begin{array}{l}{f_{0}=-\frac{1}{2}b_{0}^{2},}\\ {f_{j}=\frac{1}{2}\left[\displaystyle\sum_{k=1}^{j-1}b_{k}b_{j-k}-\frac{1}{\rho}\displaystyle\sum_{k=0}^{j-1}b_{k}b_{j-1-k}\right],j>0.}\end{array} $$ 

An analytical preprocessing for  $ \alpha \neq 0 $ is not possible here, because a relation like (2.34) is missing for that case. Fortunately this is no disadvantage here, as a check after each time step during the integration has revealed. During the whole process  $ \bar{\alpha}(u) $ evaluated due to the fitting condition turned out to be near zero. Thus the integration has been performed with fixed  $ \alpha = 0 $. The integration has been started with  $ \varphi = \Psi_{\rho,0} $,  $ \rho = 0.001 $, i.e.  $ \varphi \approx \delta_{s,1} $.
